
Projet de recommandation sportive personnalisée (HTML + Tailwind + JS)

Ce projet propose une expérience interactive de coaching sportif personnalisée, inspirée du redesign "Decathlon Coach 2025".
L’utilisateur répond à un mini questionnaire (profilage sportif) comprenant :
-Son objectif
-Son niveau
-Son lieu de pratique

Ensuite, une séance personnalisée est générée automatiquement avec :
-Un exercice ciblé
-Des instructions détaillées
-Une illustration vidéo incorporée
-Une sélection de produits Decathlon liés à la pratique (bonus)

L’ensemble est développé en HTML, TailwindCSS et JavaScript Vanilla, et offre une expérience moderne, animée et fluide.

Le projet ne nécessite aucune installation particulière.

Méthode 1 — Ouvrir directement

Téléchargez le projet ou clonez le repository
Ouvrez simplement le fichier :
index.html
Le projet fonctionne immédiatement dans votre navigateur.
