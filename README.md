<h1>Projet de Recommandation Sportive Personnalisée</h1>


<p>Ce projet propose une expérience interactive de coaching sportif personnalisée. L’utilisateur répond à un mini-questionnaire (objectif, niveau, lieu de pratique), puis une séance adaptée est générée automatiquement.</p>


<h3>1.Fonctionnalités Principales :</h3>
    <ul>
        <li>Questionnaire dynamique et intuitif</li>
        <li>Transitions fluides et validation en temps réel</li>
        <li>Génération d’un exercice personnalisé selon le profil</li>
        <li>Intégration d’une vidéo d’exécution</li>
        <li>Instructions détaillées et conseils pratiques</li>
        <li>Suggestions de produits liés à la pratique (bonus)</li>
        <li>Design responsive pensé mobile-first</li>
        <li>Interface moderne utilisant TailwindCSS</li>
    </ul>

<h3>2.Technologies Utilisées :</h3>
    <ul>
        <li>HTML5</li>
        <li>TailwindCSS</li>
        <li>JavaScript</li>
    </ul>


<h3>3.Comment Lancer le Projet :</h3>
    <h4>Aucune installation n’est nécessaire.</h4>

Méthode :
    <ul>
        <li>Télécharger ou cloner le repository</li>
        <li>Ouvrir le fichier index.html</li>
        <li>Le projet fonctionne immédiatement dans n’importe quel navigateur récent</li>
    </ul>


<h3>4.Objectifs du Projet :</h3>

<h4>Ce projet démontre :</h4>
    <ul>
        <li>La création d’un questionnaire dynamique</li>
        <li>La génération automatique d’une séance personnalisée</li>
        <li>L’intégration fluide de vidéos et illustrations</li>
        <li>L’usage d’un design moderne basé sur TailwindCSS</li>
        <li>La mise en place d’une expérience web claire, simple et responsive</li>
    </ul>
