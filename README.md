Projet de Recommandation Sportive Personnalisée


Ce projet propose une expérience interactive de coaching sportif personnalisée. L’utilisateur répond à un mini-questionnaire (objectif, niveau, lieu de pratique), puis une séance adaptée est générée automatiquement.


1.Fonctionnalités Principales :
    -Questionnaire dynamique et intuitif
    -Transitions fluides et validation en temps réel
    -Génération d’un exercice personnalisé selon le profil
    -Intégration d’une vidéo d’exécution
    -Instructions détaillées et conseils pratiques
    -Suggestions de produits liés à la pratique (bonus)
    -Design responsive pensé mobile-first
    -Interface moderne utilisant TailwindCSS


2.Technologies Utilisées :
    -HTML5
    -TailwindCSS
    -JavaScript Vanilla


3.Comment Lancer le Projet :
    -Aucune installation n’est nécessaire.

Méthode :
    -Télécharger ou cloner le repository
    -Ouvrir le fichier index.html
    -Le projet fonctionne immédiatement dans n’importe quel navigateur récent


4.Objectifs du Projet :

Ce projet démontre :
    -La création d’un questionnaire dynamique
    -La génération automatique d’une séance personnalisée
    -L’intégration fluide de vidéos et illustrations
    -L’usage d’un design moderne basé sur TailwindCSS
    -La mise en place d’une expérience web claire, simple et responsive
